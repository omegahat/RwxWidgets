0.5-0

Added
  • wxStaticBox
  • wxMenuBar
  • wxMenu
  • wxMenuItem
Support for RANGE events for which one specifies the start and end object ID as c(start, end) via the objId argument of wxObject_AddCallback

0.3-3

  • Support for the wxWindow_FindWindowBy{Name,Id,Label} methods.
  • Added support for staticMethod mechanism to allow them to be invoked as obj$Method(...) and have the obj dropped in the call to the method.
    This facilitates identifying the method, e.g. wxWindow_FindWindowByName can be called as f$FindWindowByName() without having to remember in which class it is actually defined (i.e. wxWindow).
  • AddStretchSpacer() and AddSpacer() methods for wxSizer added.

    • 0.3-2

    Added R functions for wxSlider and a slider.R example in examples/
